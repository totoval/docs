# Installation

## Server Requirements
* System: linux, macos, windows, arm...
* Cpu: 1
* Memory: 128MB
* Disk: 100MB

## Install
1. Download

    ```sh
    git clone git@github.com:totoval/totoval.git ./YOUR-PROJECT-NAME
    ```
    
2. Remove `.git` and reinit it

    ```sh
    cd ./YOUR-PROJECT-NAME
    rm -rf .git
    git init
    ```
